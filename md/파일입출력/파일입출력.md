# 파일 입출력

## I/O and Stream
I/O : 데이터의 입력과 출력
데이터는 한쪽에서 주고 한쪽에서 받는 구조
- 이때, 입력과 출력의 끝단 : 노드(Node)
- 두 노드를 연결하고 데이터를 전송할 수 있는 개념 : 스트림(Stream)
- 스트림은 단방향으로만 통신이 가능, 하나의 스트림으로 입력과 출력을 같이 처리할 수 없음
